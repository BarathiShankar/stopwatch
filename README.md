## ğŸ› ï¸ How It Works

1. **Start**:  
   - Begins/resumes the stopwatch.  
   - Uses `setInterval` to update the display every 10ms.  

2. **Stop**:  
   - Pauses the stopwatch.  
   - Preserves elapsed time for resuming later.  

3. **Reset**:  
   - Clears the timer.  
   - Resets display to `00:00:00:00`.
